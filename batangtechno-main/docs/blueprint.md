# **App Name**: Batang Techno

## Core Features:

- Entry Browsing (Viewer): Publicly display all submitted hackathon entries, including project names, descriptions, and submission links.
- Top 3 Showcase (Viewer): Highlight and display the top 3 ranked hackathon entries prominently for public viewing, without showing specific scores.
- Entry & Media Management (Admin): Allow administrators to upload, edit, and manage hackathon entries. This includes fields for team name, project school, project members (minimum three with an add button), a Google Drive video link (embedded and playable), an image upload for a thumbnail, a project description, and a dropdown for challenge chosen (SMART MOBILITY AND TRANSPORTATION, DIGITAL LITERACY AND COMBATING DISINFORMATION, TRANSPARENCY, ACCOUNTABILITY, AND GOOD GOVERNANCE, EMPLOYMENT AND ECONOMIC OPPORTUNITIES, HEALTHCARE ACCESS FOR RURAL AND REMOTE COMMUNITIES, SUSTAINABLE AGRICULTURE).
- Judge Score & Ranking Upload (Admin): Enable administrators to batch upload judge scores and finalize overall project rankings, which will then be reflected in the top 3 viewer section.
- Judging Interface (Judge): Provide a secure interface for judges to view assigned hackathon entries, access project details, and submit scores based on predefined criteria.

## Style Guidelines:

- Primary color: Deep blue (#003366) for a professional and stable feel. Secondary color: Light black / very dark grey (#222222) for backgrounds and text. Accent color: Bright azure (#3399FF) for interactive elements and highlights. Subtlety color: Medium grey (#666666) for borders and subtle details.
- Overall font: 'Inter' (sans-serif) for a modern, clean, and highly readable aesthetic across all text elements.
- Utilize minimalist, futuristic icons with a subtle glow or metallic finish, inspired by celestial bodies and spacecraft elements, for actions and navigation.
- Clean, structured layouts with subtle star-field or nebulae backgrounds. Information is presented in modular, responsive cards or panels that adapt across devices, evoking a command center interface.
- Smooth, fluid transitions reminiscent of space travel or celestial motion. Subtle glowing effects on interactive elements and data loading animations that mimic star twinkles or data streams.
